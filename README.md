# Cours Deep Learning M2

## Prérequis

- Python 3.10

## Installation

```powershell
# Créer l'environnement virtuel
py -3.10 -m venv .venv

# Activer (PowerShell)
.venv\Scripts\Activate.ps1

# Installer les dépendances
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

## Lancer Jupyter

```powershell
.venv\Scripts\Activate.ps1
jupyter lab
```

Dans Jupyter, sélectionner le kernel **Python (deep_learning_m2)**.

Les notebooks se trouvent dans le dossier `notebooks/` (voir `notebooks/00_demo.ipynb` pour vérifier que tout fonctionne).
